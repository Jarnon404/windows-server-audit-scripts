Windows Server Audit Scripts

Public-safe PowerShell audit scripts for Windows Server and domain-controller focused inventory reporting. The repository is sanitized for GitHub and designed to avoid committing customer-specific output.

Purpose

This project provides practical audit scripts for collecting Windows Server inventory, operational state, security-related configuration and domain-controller focused information into reviewable reports.

Safety model

No generated reports, customer names, tenant identifiers, internal hostnames, private IP addresses, usernames, email addresses or credentials should be committed to this repository.
